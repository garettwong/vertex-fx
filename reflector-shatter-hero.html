Glass reflector portfolio pattern

A hero that breaks into its next case.

A scratch-built, asset-free version of a glossy portfolio landing effect: the main panel behaves like a reflective acrylic object, then bursts into glassy fragments before reforming with the next project category.

Implementation notes

Nocturne

A fictional campaign surface rendered with CSS gradients, not photos.

Reflector depth without external 3D assets.

01 / material

Acrylic from CSS layers

Specular sweeps, radial highlights, backdrop blur and gradient scenes sell a reflective surface without shipping proprietary imagery.

02 / transition

Fragments are generated

Shards are temporary DOM elements with randomized directions and rotations, so every category swap feels like glass reassembling.

03 / safe rebuild

Portfolio-ready but generic

The demo uses fictional case names, procedural colors and no copied photos, code, logos, model files or shader materials.
